# 우리 동네 폭염 대응 AI

기상청 폭염 영향예보 일자료를 기반으로 지역별 폭염 위험을 시뮬레이션하는 Streamlit 웹 앱입니다.

CO₂ 농도, 해수면 온도 상승, 습도 변화, 도시화·열섬 강도, 대응 수준을 조정하면 건강 위험도, 전력 사용 위험도, 우리나라 위험 지도, AI 해설 박스가 함께 갱신됩니다.

## 주요 기능

- 2025년 5월~9월 폭염 영향예보 일자료 기반 분석
- 지역별 폭염일수, 열대야, 최고체감온도, 습도 통계 활용
- 시나리오 변화에 따른 건강 위험도·전력 위험도 계산
- 위험도 게이지와 우리나라 지도 시각화
- 선택 지역의 실제 관측 타임라인 제공
- 학생, 어르신, 야외근로자, 전력관리 대상별 대응 추천

## 실행 방법

1. 저장소를 내려받습니다.

```bash
git clone <repository-url>
cd <repository-name>
```

2. 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

3. Streamlit 앱을 실행합니다.

```bash
streamlit run app.py
```

4. 브라우저에서 표시되는 로컬 주소로 접속합니다.

```text
http://localhost:8501
```

## 데이터 안내

`data/` 폴더에는 기상청 폭염 영향예보 일자료가 들어갑니다. 원본 파일은 `.xls` 확장자를 사용하지만, 실제로는 `cp949` 인코딩의 탭 구분 텍스트 형식이므로 앱에서 자동으로 읽고 정리합니다.

앱은 같은 날짜와 지점이 반복된 행을 중복 제거한 뒤 지역별 통계를 계산합니다.

## 사용 데이터 컬럼

- 일시
- 지점
- 폭염여부(O/X)
- 최고체감온도(°C)
- 최고기온(°C)
- 평균기온(°C)
- 최저기온(°C)
- 평균상대습도(%)
- 폭염특보(O/X)
- 폭염영향예보(단계)
- 열대야(O/X)
- 자외선지수(단계)

## 주의

이 앱은 교육 및 시연용 시뮬레이터입니다. 실제 재난 대응, 보건 판단, 행정 의사결정에는 공식 기상 정보와 검증된 예측 모델을 함께 사용해야 합니다.
